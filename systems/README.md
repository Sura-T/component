# Individual Systems Directory

Each team member has two individual systems:

- `systems/<member>/system-a`
- `systems/<member>/system-b`

These systems must focus on configuration and assembly only. Shared logic should be imported from:

- `@monorepo/ui-components`
- `@monorepo/utils`
- `@monorepo/feature-x`
- `@monorepo/feature-y`
